Información General

Objetivos del OVI

Objetivo general:

Fortalecer los conocimientos necesarios para el diseño de páginas Web utilizando HTML5, CSS3 y el software del control de versiones GIT.


  • Objetivo específico 1: Comprender la estructura básica de HTML y diseñar un sitio Web con un encabezado, pie de página y un menú de navegación aplicando el modelo de caja.
  • Objetivo específico 2: Aplicar estilos CSS para mejorar la apariencia del sitio web.
  • Objetivo específico 3: Desplegar un sitio web básico creado con HTML5 y CSS3 en Github pages.

Objetivos del Curso

  • > Comprender la función del sofware para control de versiones GIT

  • > Diseñar y crear el contenido de un sitio Web Funcional

  • > Codificar y poner en funcionamiento un sitio web con tematicas asociadas al curso

  • > Evaluar la calidad y funcionamiento del sitio web creado

  • > Crear el plan de mejoramiento para el sitio web diseñado

Tematicas

  • >Estructura Básica de HTML

  • >Estilos básicos CSS

  • >Subir sitio a github pages

  • >Software para gestión de repositorios

  • >Ramas de GIT
